# Contributing to code-formatter

Thank you for your interest in contributing! We appreciate your help.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/code-formatter.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature`

## Development

```bash
# Watch mode (auto-compile TypeScript)
npm run dev

# Run tests
npm test

# Run linter
npm run lint
```

## Submitting Changes

1. Make your changes and commit them
2. Push to your fork
3. Create a Pull Request against `main`
4. Ensure all CI checks pass

## Code of Conduct

Be respectful to other contributors. We value diverse perspectives and welcome all contributors.

## Questions?

Open an issue or discussion if you have questions!
